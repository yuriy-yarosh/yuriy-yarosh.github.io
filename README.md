# yarosh.dev

Personal site, blog, and project portfolio of Yuriy Yarosh.

Live at **[yarosh.dev](https://yarosh.dev)**

## Tech Stack

- **Framework**: [TanStack Start](https://tanstack.com/start) + [TanStack Router](https://tanstack.com/router) (file-based routing, SSG prerendering)
- **Build**: [Vite](https://vite.dev/) + [Nitro](https://v3.nitro.build/) (node-server preset)
- **Rendering**: React 19 with SSR / SSG prerendering
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) v4
- **3D**: [react-three-fiber](https://r3f.docs.pmnd.rs/) + [drei](https://drei.docs.pmnd.rs/) + [postprocessing](https://postprocessing.docs.pmnd.rs/)
- **i18n**: [Paraglide JS](https://inlang.com/m/gerre34r/library-paraglide) — 14 locales
- **Content**: [content-collections](https://www.content-collections.dev/) with MDX
- **Testing**: [Vitest](https://vitest.dev/) + [Storybook](https://storybook.js.org/)
- **Linting**: [Biome](https://biomejs.dev/)
- **Package Manager**: Yarn 4 (Corepack)
